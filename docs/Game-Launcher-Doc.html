01

Visão Geral

Game Launcher é um aplicativo desktop pessoal estilo Steam que permite organizar, descobrir e lançar jogos de todas as plataformas em um só lugar.

Scanner Automático

Detecta jogos do Steam, Epic, Xbox, GOG e pastas comuns automaticamente

Biblioteca Visual

Grid ou lista com capas, busca e filtros por plataforma

Play Time Tracking

Registra tempo de jogo por sessão automaticamente

Stellar Blade

Parser de save para extrair conquistas sem Steam API

05

Banco de Dados

Localização: %APPDATA%\game-launcher\game-launcher.db

Tabelas

games

CREATE TABLE games ( id TEXT PRIMARY KEY, name TEXT NOT NULL, executable_path TEXT NOT NULL, cover_image TEXT, platform TEXT NOT NULL DEFAULT 'other', last_played TEXT, play_time INTEGER NOT NULL DEFAULT 0, last_session_start TEXT, steam_app_id TEXT, created_at TEXT NOT NULL DEFAULT (datetime('now')), updated_at TEXT NOT NULL DEFAULT (datetime('now')) );

achievements

CREATE TABLE achievements ( id INTEGER PRIMARY KEY AUTOINCREMENT, game_id TEXT NOT NULL, apiname TEXT NOT NULL, achieved INTEGER NOT NULL DEFAULT 0, unlocktime INTEGER NOT NULL DEFAULT 0, name TEXT NOT NULL, description TEXT, icon TEXT, icongray TEXT, updated_at TEXT NOT NULL DEFAULT (datetime('now')), FOREIGN KEY (game_id) REFERENCES games(id) ON DELETE CASCADE, UNIQUE(game_id, apiname) );

play_sessions

CREATE TABLE play_sessions ( id INTEGER PRIMARY KEY AUTOINCREMENT, game_id TEXT NOT NULL, start_time TEXT NOT NULL, end_time TEXT, duration_minutes INTEGER NOT NULL DEFAULT 0, FOREIGN KEY (game_id) REFERENCES games(id) ON DELETE CASCADE );

Play Time Tracking

O app registra last_session_start ao abrir um jogo. Na próxima vez que o jogo for aberto (ou o app fechar), calcula o tempo decorrido e salva em play_time (minutos).

const url = `https://store.steampowered.com/api/storesearch/?term=${name}&l=english&cc=US`; return { coverImage: `https://cdn.akamai.steamstatic.com/steam/apps/${appId}/header.jpg`, steamAppId: appId };

Filtros

XBOX_EXCLUDE

~170 nomes exatos de apps não-jogos (Microsoft, Spotify, Discord, etc.)

SKIP_EXE_NAMES

uninstall, setup, install, config, settings, etc.

SKIP_DIRS

redist, vcredist, .git, node_modules, src, build, etc.

MODS_MAP

LOOT, CurseForge, Vortex, Mod Organizer, Creation Kit, etc.

Xbox: Nomes de Pacote ≠ Nome do Jogo

O pacote Microsoft.ForteBaseGame NÃO é o nome do jogo. O parser lê AppxManifest.xml para extrair o <DisplayName> real.

Guia de Manutenção

Adicionar nova plataforma de scanner

1. Criar scanNovaPlataforma() em gameScanner.ts 2. Retornar ScannedGame[] 3. Adicionar em scanAllGames() com Promise.all 4. Adicionar caso em getGameCount() em database.ts 5. Adicionar cor/label em GameCard.tsx e GameDetail.tsx

Adicionar novo IPC handler

1. Handler em main.ts: ipcMain.handle('nome', ...) 2. Método em preload.ts: ipcRenderer.invoke('nome', ...) 3. Tipo em types/index.ts: ElectronAPI interface 4. Usar: window.electronAPI.nome()
